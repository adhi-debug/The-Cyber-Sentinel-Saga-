AI THREAT DETECTION PROJECT
LOCAL RUN & TEST GUIDE

====================================================

1. START POSTGRES + REDIS
   ====================================================

From project root:

cd G:\PROJECT\temp\ai-threat-detection

docker compose -f dev.compose.yml up postgres redis

Wait until both services show healthy.

====================================================
2. START BACKEND
================

Open Terminal 2:

cd G:\PROJECT\temp\ai-threat-detection\backend

activate venv

..venv\Scripts\activate

Start FastAPI:

python -m uvicorn app.main:app --reload

Expected:

INFO: Uvicorn running on http://127.0.0.1:8000

Verify:

curl http://127.0.0.1:8000/health

Expected:

{
"status":"healthy"
}

====================================================
3. START FRONTEND
=================

Open Terminal 3:

cd G:\PROJECT\temp\ai-threat-detection\frontend

pnpm install

pnpm dev

Expected:

http://localhost:5173

Open browser:

http://localhost:5173

====================================================

======================

curl http://127.0.0.1:8000/health
